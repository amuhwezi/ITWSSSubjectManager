The Subject Manager helps you to add, edit and delete the subjects on a schools curriculum. This may contribute to generating a final academic report for Students. 

== Description ==

The Subject Manager plugin is a simple to use plugin that make use of shortcode. The following shortcode(s) are available;
[form] - displays the innitial form for you to enter some subject information.
[show] - to display the data you will have entered into the database, you may activate this short-code.
[comb] - incase you would like to add a combination of subjects, you may also add this shortcode that presesnts a form to enter required details about the subject combinbiton. Note that this short code is dependant on the previous one to be running as well.
[dispcomb] - this displays the added subject combinations and enables for editing.

== Installation ==
1. Upload the plugin folder to the plugins directory in your wordpress directory '/wp-content/plugins/'
2. Refresh your plugins page to show the Subject Manager plugin.
3. Activate the plugin through the menu right below the plugin name.
4. Create a new page and in the contents type shortcode [form] and publish.
5. Create another page and type shortcode [show] and publish.
6. You may add other pages as well for more functionality.
